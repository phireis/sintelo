CONCEITOS BÁSICOS



Para melhor compreensão dos assuntos abordados, serão apresentados nesta seção conceitos gerais sobre as linguagens que servirão para fundamentar o estudo de todos os tipos de linguagens:



Símbolos: são os elementos que compõem uma linguagem. Letras e dígitos são exemplos de símbolos freqüentemente usados.



Alfabeto: defini-se com sendo um conjunto de símbolos finito e não-vazio (HOPCROFT, ULLMAN, MOTWANI, 2003). Segundo MENEZES (2001) alfabeto é uma entidade abstrata básica, não sendo definida formalmente. Normalmente é utilizado o símbolo  para representar o alfabeto.

Exemplos de alfabetos:

 = {0,1) – alfabeto binário

 = {a, b, ...., z} – conjunto de todas letras minúsculas.

Na Linguagem de Programação como o Pascal, por exemplo, o alfabeto é todo o conjunto de símbolos usados na construção do programa, incluindo letras, dígitos, caracteres especiais como “>”, “/” e espaço ou branco (MENEZES, 2001).



Palavra: é uma seqüência finita de símbolos escolhidos de um determinado alfabeto (HOPCROFT, ULLMAN, MOTWANI, 2003).

Exemplo de palavras:

Se temos o  = (a,b,c), então abcb é uma palavra deste alfabeto.

Prefixo de uma palavra é o número qualquer de símbolos tomados de seu inicio, já sufixo é o número qualquer tomado de seu fim. Podemos citar o seguinte exemplo sobre prefixo e sufixo:

Tendo a seguinte palavra abcb sobre o  = (a,b,c) temos:

Na Linguagem de programação como o Pascal, por exemplo, uma palavra é um programa (MENEZES, 2001).

Linguagem: é um conjunto de cadeias de símbolos que recebem de algum alfabeto. Hopcroft, Ullman e Motwani (2003) afirmam que linguagem é um conjunto de palavras, todos escolhidos a partir de um *, onde  é um alfabeto especifico. Se  é um alfabeto, e L  *, então L é uma linguagem sobre .
