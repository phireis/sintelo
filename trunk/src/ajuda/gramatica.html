GRAMÁTICA



Menezes (2001) define como sendo, um conjunto de regras a quais, quando aplicados sucessivamente, geram palavras, e o conjunto destas define uma linguagem. As gramáticas são instrumentos que facilitam muito a definição das características sintáticas das linguagens e permitem definir de forma formal e sistemática, uma representação finita para linguagens infinitas.

Furtado (2001) defini Gramática, de maneira informal como sendo:

  • Um sistema gerador de linguagens;

  • Um sistema reescrita;

  • Uma maneira finita de descrever (representar) uma linguagem;

  • Um dispositivo formal usado para especificar de maneira finita e precisa uma linguagem potencialmente infinita.

Exemplo intuitivo de uma Gramática:

<sentença> :: = <sujeito> <predicado>

<sujeito> :: = <substantivo>

| <artigo> <substantivo>

| <artigo> <adjetivo> <substantivo>

<predicado> :: = <verbo> <objeto>

<substantivo> :: = joão | Maria | cachorro | livro | pão

<artigo> :: = o | a

<adjetivo> :: = pequeno | bom | bela

<verbo> :: = morde | le | olha

<objeto> :: = <substantivo>

| <artigo> <substantivo>

| <artigo> <adjetivo> <substantivo>



Os conjuntos finitos que uma gramática possui são variáveis (ou denominadas não terminais), assim a linguagem gerada é definida recursivamente em termos das variáveis e de símbolos primitivos chamados terminais, que pertencem ao alfabeto da linguagem.

Tipos de gramáticas



Chomsky classificou as gramáticas conforme as restrições na forma de produção, esta é conhecida como Hierarquia de Chomsky. Os tipos de gramáticas definidos nesta hierarquia são:

Gramática tipo 0 (gramática sem restrição): Não há restrição na forma de produção. As regras de uma gramática tipo 0 são regras da forma α → β. Exemplo:

S→ ABS | ab

Ba→ aB | a

Aa → aa | a | ε



Gramática tipo 1 (gramáticas sensíveis ao contexto): As gramáticas tipo 1 são as gramáticas com as regras da forma, que o símbolos inicial não apareça do lado direito de nenhuma produção e para cada produção α → β, em que exige | α | ≤ | β|, com exceção da regra S → ε. Exemplo:



S → aBC| aABC

A → aABC| aBC

aB → ab aB → ab

Gramática tipo 2 (gramática livre de contexto): São as gramáticas com regras da forma A → β, onde A é um símbolo não terminal, e β é uma seqüência qualquer de V*, possivelmente vazia. Exemplo:

S → aSb| A

A → cAd| e



A gramática do tipo 2 não e definido como restrição do tipo 1, porque se permite produções da forma A → ε, com A ≠ S, e também permite que o símbolo inicial apareça do lado direito das produções.

Gramática tipo 3 (gramáticas regulares): As gramáticas do tipo 3 só podem ter regras das formas:



A → a B, onde A e B são não terminais e a é um terminal;

A → a, onde A é um não terminal, e a é um terminal;

A → ε, onde A é um não terminal.
