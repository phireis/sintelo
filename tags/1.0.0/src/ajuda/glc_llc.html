GRAMÁTICAS E LINGUAGENS LIVRES DE CONTEXTO



Gramáticas livres de contexto (GLC), como classificadas por Chomsky, são aquelas que definem linguagens livres de contexto, que compreendem a maioria das linguagens de programação existentes. Elas definem linguagens que não podem ser representadas com expressões regulares, como encadeamento de abertura e fechamento de parênteses (HOPCROFT; ULLMAN; MOATWANI, 2003).

A sintaxe de uma linguagem define a sua estrutura hierárquica, e pode ser representada por gramáticas livres de contexto. Gramáticas apresentam vantagens tanto para os projetistas de uma linguagem quanto para desenvolvedores de compiladores, entre elas (AHO; SETHI; ULLMAN, 1995):

  1. Uma gramática proporciona uma especificação sintática precisa, e ao mesmo tempo fácil de entender, de uma linguagem;

  2. É possível gerar analisadores sintáticos automaticamente a partir de certas gramáticas;

  3. Erros, como ambigüidade, por exemplo, podem ser encontrados de maneira automatizada através da definição da linguagem utilizando gramáticas;

  4. As linguagens podem ser facilmente aprimoradas quando já existe uma implementação de uma linguagem baseada em uma gramática.

Gramáticas são a forma mais utilizada, mais flexível e mais eficaz de se representar as linguagens, principalmente as livres de contexto.



COMPONENTES DE UMA GLC



Uma GLC pode ser definida como G = {N, T, P, S}, e suas componentes são:

  1. um conjunto de terminais (T): define as strings da linguagem definida pela gramática;

  2. um conjunto de não-terminais (N): define as variáveis que podem representar diferentes seqüências de terminais. “Cada variável representa uma linguagem, isto é, um conjunto de strings” (HOPCROFT; ULLMAN; MOTWANI, 2003, p. 182);

  3. símbolo inicial (S): variável que representa a linguagem que está sendo definida;

  4. conjunto de produções ou regras (P): representam a definição recursiva de uma linguagem, sendo formada por uma variável, no lado esquerdo, e uma seqüência de símbolos terminais ou não-terminais do lado direito do símbolo de produção, que varia de acordo com a forma de representação da gramática.
