EXPRESSÕES REGULARES



Cada expressão regular r denota uma linguagem L (r), onde as regras de definição especificam como a linguagem é formada através da combinação, em varias formas, das linguagens denotadas pelas subexpressões de r (AHO; SETHI; ULLMANN, 1995).

A representação da linguagem L (r) pode ser definida a partir de expressões básicas como (PRICE; TOSCANI, 2001):

  1. Ø representa a linguagem vazia (conjunto contendo zero palavra);

  2. { representa a linguagem cuja única palavras é a vazia;

  3. { x | x representa a linguagem cuja única palavras é x;

  4. se r1 | r2 são expressões regulares definindo as linguagens L (r1) e L (r2), tem-se que:

    1. r1 | r2 é a linguagem cujas palavras constituem o conjunto L (r1)  L (r2);

    2. r1 r2 é a linguagem { vw | v L (r1) e w L (r2), isto é, a linguagem cujas palavras são formadas pela concatenação de uma palavra de L(r1) com uma palavras de L(r2), nesta ordem;

    3. r1* representa o conjunto L*(r1), isto é, o conjunto de palavras que podem ser formadas concatenando-se zero ou mais palavras de L(r1).

Exemplo de expressões regulares, seja  = {a, b}(AHO; SETHI; ULLMANN, 1995):

  1. a expressão regular a | b significa o conjunto {a,b};

  2. a expressão regular (a | b) (a | b) denota {aa, ab, ba, bb), o conjunto de todas as cadeias de a's b's de comprimento dois. Outra expressão regular para esse conjunto é aa | ab | ba | bb.

  3. A expressão regular a* significa o conjunto de todas as cadeias de zero ou mais a's, isto é, {a, aa, aaa, ....};

  4. A expressão regular (a | b)* significa o conjunto de todas as cadeias contendo zero ou mais instâncias de um a ou um b, ou seja, junto de todas as cadeias a's e b's.
